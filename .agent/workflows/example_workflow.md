This is an example workflow file! You can create files like this in the `.agent/workflows/` directory.

Whenever you ask me to perform a complex, multi-step task, I will look for files in this folder to learn the exact steps I need to take. 

### Structure of a Workflow
1. Start with the YAML frontmatter at the top (the text between the `---` lines) to give the workflow a description.
2. Outline the exact steps you want me to take down here.

You can create more workflows in this folder, for example `adding_new_truss.md`, or create a `skills/` folder instead if you prefer the Skills structure.
